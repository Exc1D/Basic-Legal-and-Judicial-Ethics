# Basic Legal and Judicial Ethics

An Obsidian vault for Philippine **Basic Legal and Judicial Ethics**. The vault is organized around doctrine notes, CPRA canons, judicial ethics, notarial practice, cases, bar traps, flashcards, tables, practical exercises, and source tracking.

## Folder structure

```text
Basic Legal and Judicial Ethics/
├─ 00 Dashboard/              # Home dashboards and MOCs
├─ 01 Core Doctrines/         # Foundational doctrine notes
├─ 02 CPRA/                   # Code of Professional Responsibility and Accountability
├─ 03 Judicial Ethics/        # New Code of Judicial Conduct and judge discipline
├─ 04 Notarial Practice/      # Notarial rules, registers, defects, remote notarization
├─ 05 Cases/                  # Case digests and case index
├─ 06 Bar Traps/              # Common exam traps and distinctions
├─ 07 Flashcards/             # Active recall notes

├─ 09 Practical Exercises/    # Drafting exercises and templates

```

## Note standard

Each major doctrine note should contain:

1. Rule
2. Elements or requirements
3. Exceptions or limits
4. Related cases
5. Bar trap
6. Recall questions
7. Source ladder

## Copyright-safe rule

Do not commit copyrighted PDFs, screenshots, copied reviewer tables, or verbatim review-center text to this public repo. Use review materials only as private reading guides, then write original summaries from official rules, cases, and public primary sources.
