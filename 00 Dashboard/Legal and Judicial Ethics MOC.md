## Daily study flow

1. Pick one active build item.
2. Create or improve its doctrine note.
3. Link at least one case.
4. Add one bar trap.
5. Add 3 recall questions.
6. Move `status: seed` to `status: reviewed` only when the note is usable for bar review.

## Copyright-safe workflow

Use official public sources where possible: Supreme Court issuances, Lawphil, rules, statutes, bar bulletins, and official circulars. Treat review materials as private references only. Do not commit copyrighted PDFs, screenshots, copied reviewer tables, or verbatim review-center text to this public repository.
