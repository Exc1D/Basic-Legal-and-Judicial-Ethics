# Source Index

Use this folder to track official and secondary sources used in the vault.

## Source priority

1. 1987 Constitution
2. Rules of Court, especially Rule 138 and Rule 139-A
3. Supreme Court administrative matters and circulars
4. Code of Professional Responsibility and Accountability
5. Lawphil and Supreme Court case texts
6. Statutes such as RA 7662, RA 9225, RA 6713, RA 7160, and RA 910
7. Bar bulletins and official syllabi
8. Secondary reviewers, used only as private guides

## Copyright-safe workflow

- Do not upload copyrighted PDFs to the repo.
- Do not copy reviewer tables or model answers verbatim.
- Use reviewers only to identify topics, then rewrite from primary sources.
- Every doctrine note should eventually link to an official rule or case source.
- Every case note should eventually contain a public source link.
